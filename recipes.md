# recipes.md — 바로 써먹는 작업 예시

지금 필요한 결과물에 가장 가까운 작업 하나를 골라 시작합니다. 순서는 없습니다.

---

## 파일 정리

다운로드 폴더처럼 너무 더러워진 곳을 자동으로 정돈합니다.

예시:

- 다운로드 폴더의 파일을 종류(PDF·이미지·ZIP)별로 분류하기
- 파일명 앞에 오늘 날짜 일괄 붙이기
- 강의 자료가 섞인 폴더를 과목별로 나누기
- 같은 이름이 반복되는 파일 찾기

Codex에게 요청할 말:

```text
/build 다운로드 폴더의 파일을 종류별로 폴더 만들어서 옮겨줘
```

---

## 문서 정리

마크다운, 한글·Word 문서, 노트를 모아 보기 좋게 만듭니다.

예시:

- 팀플 회의록 여러 개에서 결정사항과 할 일만 뽑기
- 강의 노트를 주차별 인덱스로 만들기
- 긴 보고서를 섹션 요약으로 줄이기
- 조사한 내용을 표·체크리스트로 정리하기

Codex에게 요청할 말:

```text
/build 팀플 회의록 여러 개를 모아서 결정사항이랑 할 일만 정리해줘
```

`.docx` 파일을 다루면 `docx` 스킬이 자동으로 활성화됩니다.

---

## 데이터 처리

CSV, 엑셀, 설문 응답 같은 표 데이터를 정리합니다.

예시:

- 설문조사 응답의 비율·평균 보기
- CSV 여러 개를 하나로 합치기 (열 이름이 달라도)
- 중복 행·빈 셀 찾기
- 날짜별로 데이터를 분리해 파일로 저장하기

Codex에게 요청할 말:

```text
/build 설문조사 CSV를 읽어서 항목별 응답 분포를 표로 보여줘
```

메모:

- `xlsx` 스킬은 기본 폰트로 Arial을 사용합니다. 한글이 들어간 결과물은 "맑은 고딕"으로 바꿔달라고 지시하세요.
- `xlsx` 스킬에 있는 재무 모델 색상 규약(Blue/Black/Green/Red/Yellow)은 학교 과제·일반 표 작업에서는 무시해도 됩니다.

---

## 웹 자료 모으기

공지·기사·자료를 자동으로 가져와 파일로 저장합니다. (이전의 "웹 자료 수집"과 "웹 스크래핑"을 합친 카테고리)

예시:

- 학교·학과 공지 페이지에서 제목·날짜·링크 수집
- 관심 있는 블로그 RSS 피드에서 최신 글 모으기
- 특정 사이트의 공개 표를 CSV로 저장
- 여러 페이지를 순회하며 데이터 쌓기

Codex에게 요청할 말:

```text
/build 학과 공지 페이지에서 제목·날짜·링크를 매주 자동으로 가져와줘
```

`web-scraper` 스킬이 정적·동적 페이지를 구분해 적절한 코드를 만듭니다.

메모:

- 사이트의 robots.txt와 이용약관을 먼저 확인합니다.
- 요청 간격은 최소 1초 이상 두어 서버에 부담을 주지 않습니다.

---

## 보여줄 화면 만들기

결과를 브라우저에서 바로 볼 수 있는 HTML 페이지로 만듭니다. (이전의 "작은 화면 만들기"와 "웹 서비스/앱 개발"을 합친 카테고리)

예시:

- CSV 결과를 검색·정렬 가능한 표로 보여주기
- 정리한 메모들을 한 페이지 인덱스로 묶기
- 그래프와 통계를 담은 단일 HTML 보고서
- 입력란이 있는 작은 도구 (단위 변환기, 점수 계산기 등)

Codex에게 요청할 말:

```text
/build 내 작업 결과를 한 페이지 HTML로 보기 좋게 정리해줘
```

Flask·외부 호스팅은 처음 단계에서 쓰지 않습니다. 한 파일짜리 HTML로 시작합니다.

---

## 데이터 분석

표 데이터에서 패턴을 찾고 그래프로 봅니다.

예시:

- 설문 응답을 항목별 막대 그래프로 그리기
- 시간에 따른 변화(매출·방문자·점수)를 선 그래프로 보기
- 그룹별 평균 비교하기
- 빈 셀·잘못된 값 찾기

Codex에게 요청할 말:

```text
input/survey.csv를 보고 항목별 응답을 막대 그래프로 그려줘
```

결과를 위 "보여줄 화면 만들기"와 결합하면 HTML 보고서가 됩니다.

---

## 공모전·교과목 프로젝트

기획부터 발표까지 한 폴더로 정리합니다.

예시:

- 공모전 아이디어를 기획서 초안(.docx)으로 정리
- 팀 프로젝트 폴더 구조 만들기 (역할·일정 포함)
- 발표 스크립트와 슬라이드 초안(.pptx)
- 예상 질문과 답변 정리
- 제출 전 체크리스트 확인

Codex에게 요청할 말:

```text
/build "AI 활용 교육 공모전" 폴더 만들고 docx 스킬로 기획서 초안, pptx 스킬로 발표 초안까지 잡아줘. 마감은 7월 31일이야.
```

`docx`/`pptx` 스킬이 자동으로 활성화됩니다. 한글 폰트가 깨지면 "맑은 고딕으로 바꿔줘"라고 지시하세요.

## 내 작업 루틴 만들기 (선택)

이 워크스페이스에 익숙해진 뒤, 자주 반복하는 흐름을 저장합니다.

예시:

- 자주 쓰는 README·보고서 템플릿 만들기
- 새 `/명령어` 플레이북 만들기
- 작업 전후 체크리스트 만들기

Codex에게 요청할 말:

```text
내가 자주 반복하는 작업을 물어보고, 이 워크스페이스에 플레이북으로 저장해줘.
```
