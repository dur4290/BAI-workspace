# recipes.md — 바로 써먹는 작업 예시

무엇을 만들지 막막할 때 고르는 작업 예시입니다. 순서대로 따라갈 필요는 없습니다. 지금 필요한 결과물에 가장 가까운 것을 하나 고르면 됩니다.

---

## 파일 정리

반복되는 파일 작업을 자동화합니다.

예시:

- 폴더 안 파일 목록을 CSV로 저장하기
- 파일명에 날짜 붙이기
- 여러 텍스트 파일을 하나로 합치기
- 다운로드 폴더 정리 preview 만들기

Codex에게 요청할 말:

```text
/build 폴더 안 파일 목록을 CSV로 저장하는 자동화 만들기
```

---

## 문서 정리

마크다운, 텍스트, 표를 보기 좋게 정리합니다.

예시:

- 회의 메모를 액션 아이템으로 정리하기
- 긴 텍스트를 섹션별 요약으로 바꾸기
- 폴더 안 문서를 인덱스로 만들기
- 조사 내용을 표와 체크리스트로 정리하기

Codex에게 요청할 말:

```text
/plan 여러 메모 파일을 하나의 정리 문서로 합치고 싶어
```

---

## 데이터 처리

CSV나 Excel처럼 반복해서 만지는 데이터를 정리합니다.

예시:

- CSV 열 이름 정리하기
- 중복 행 찾기
- 날짜별 결과 파일 만들기
- 간단한 요약 통계 만들기

Codex에게 요청할 말:

```text
/build CSV 파일을 읽어서 필요한 열만 남기는 도구 만들기
```

---

## 웹 자료 수집

공개 웹페이지에서 필요한 정보를 가져와 파일로 저장합니다.

예시:

- 공지 페이지 제목과 링크 모으기
- 공개 표를 CSV로 저장하기
- RSS 피드에서 최근 글 목록 만들기
- 웹페이지 본문을 마크다운으로 저장하기

Codex에게 요청할 말:

```text
/plan 공개 웹페이지에서 제목과 링크를 CSV로 저장하고 싶어
```

---

## 작은 화면 만들기

결과를 HTML이나 간단한 화면으로 확인합니다.

예시:

- 단일 HTML 보고서 만들기
- CSV 요약 결과를 표로 보여주기
- 검색 가능한 메모 페이지 만들기
- 간단한 로컬 대시보드 만들기

Codex에게 요청할 말:

```text
/build CSV 결과를 브라우저에서 볼 수 있는 HTML 표로 만들기
```

---

## 내 작업 루틴 만들기

자주 반복하는 흐름을 이 워크스페이스 안에 저장합니다.

예시:

- 자주 쓰는 보고서 템플릿 만들기
- 반복 작업용 README 템플릿 만들기
- 새 `/명령어` 플레이북 만들기
- 작업 전후 체크리스트 만들기

Codex에게 요청할 말:

```text
내가 자주 반복하는 작업을 물어보고, 이 워크스페이스에 플레이북으로 저장해줘.
```

```text
현재 워크스페이스 상태를 보고, 오늘 바로 만들 수 있는 작은 작업 하나를 추천해줘.
```
