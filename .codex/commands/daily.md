# /daily — 오늘 작업 기록

오늘 한 일, 막힌 점, 다음 액션을 짧게 남깁니다.

## 저장 위치

`20-areas/daily/YYYY-MM-DD.md`

같은 날짜 파일이 있으면 아래에 이어 씁니다.

## 실행 순서

### 1단계: 작업 맥락 확인

가능하면 Codex가 직접 확인합니다.

- 최근 Git 변경
- 오늘 수정한 `10-projects/` 파일
- 새로 만든 `00-inbox/`, `20-areas/`, `30-resources/` 파일

### 2단계: 질문

질문은 하나씩 묻습니다.

```text
Q1/5 — 오늘 무엇을 만들거나 시도했나요?
```

```text
Q2/5 — 가장 잘 된 것은 무엇인가요?
```

```text
Q3/5 — 막힌 점이나 헷갈린 점은 무엇인가요?
```

```text
Q4/5 — 다음에 할 작은 행동 1개는 무엇인가요?
```

```text
Q5/5 — 나중에 다시 볼 만한 팁이나 배움이 있나요?
```

### 3단계: 기록 저장

아래 형식으로 저장합니다.

```markdown
## 작업 기록 - HH:mm

### 오늘 한 것

### 잘 된 것

### 막힌 점

### 다음 액션

### 남길 메모
```
