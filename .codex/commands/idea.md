# /idea — 아이디어 메모

떠오른 아이디어를 빠르게 메모로 남깁니다.

## 사용법

```text
/idea [아이디어 제목 또는 내용]
```

## 실행 순서

1. 오늘 날짜를 KST 기준으로 확인합니다.
2. `00-inbox/ideas/` 폴더가 없으면 만듭니다.
3. 파일 이름은 `YYYY-MM-DD-아이디어제목.md`로 만듭니다.
4. 비슷한 아이디어가 이미 있으면 새로 만들기 전에 알려줍니다.
5. 저장 경로를 알려줍니다.

## 파일 템플릿

```markdown
---
date: YYYY-MM-DD
status: 씨앗
---

# 아이디어 제목

## 한 줄 설명

## 왜 만들고 싶은가?

## 어떻게 작게 시작할까?

## 참고할 것

---
status: 씨앗 -> 자라는 중 -> 완성 -> 보관
```
