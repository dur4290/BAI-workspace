# project-finish-checklist.md — 프로젝트 완료 전 점검표

학생 작업을 공유하거나 제출하기 전에 아래 항목을 확인합니다.

## 실행

- [ ] README만 보고 실행할 수 있다.
- [ ] 필요한 도구와 버전이 적혀 있다.
- [ ] 샘플 입력 또는 준비 방법이 있다.
- [ ] 결과물을 어디에서 확인하는지 적혀 있다.

## 결과

- [ ] 최소 기능이 실제로 동작한다.
- [ ] 출력 파일, 화면, 표, 문서 중 하나로 결과가 남는다.
- [ ] 실패했을 때 원인을 알 수 있는 메시지가 있다.
- [ ] 불필요한 큰 기능이나 미완성 기능을 완료처럼 적지 않았다.

## 기록

- [ ] 무엇을 만들었는지 한 줄로 설명할 수 있다.
- [ ] 한계와 다음 개선점을 적었다.
- [ ] 사용한 입력 데이터가 안전한 예시인지 확인했다.
- [ ] 개인 PC 절대경로나 개인 정보가 들어 있지 않다.

## 보안

- [ ] 실제 API 키, 토큰, 비밀번호가 없다.
- [ ] `.env` 파일은 포함하지 않았다.
- [ ] 외부 서비스에 쓰기 작업이 있다면 사용자 확인 절차가 있다.
