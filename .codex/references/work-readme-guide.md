# work-readme-guide.md — 작업 README 작성 기준

새 작업의 `README.md`는 긴 소개 문서가 아니라 다음 세션에 바로 이어서 실행할 수 있는 작업 카드입니다.

## 필수 섹션

```markdown
# 작업 이름

## 한 줄 목표

## 작업 카드

목표:
입력:
출력:
성공 기준:
오늘 만들 최소 버전:

## 만들 기능

## 실행 방법

## 확인 방법

## Codex에게 다음에 요청할 말

## 메모
```

## 작성 기준

- 입력과 출력이 눈에 보이게 씁니다.
- 첫 버전은 1시간 안에 확인 가능한 크기로 줄입니다.
- 실행 방법은 사용자가 그대로 복사할 수 있게 씁니다.
- 확인 방법은 "무엇을 열어보면 성공인지"까지 씁니다.
- 외부 API, 로그인, 배포는 기본안으로 넣지 않습니다.
- 예시 파일이 필요하면 `input/`에 작고 안전한 샘플을 둡니다.
