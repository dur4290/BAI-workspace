# work-readme-guide.md — 작업 README 작성 기준

새 작업의 `README.md`는 소개 문서가 아니라 다음 세션에 바로 이어서 작업할 수 있는 실행 계획서입니다.

## 필수 섹션

```markdown
# 작업 이름

## 한 줄 목표

## 왜 만들까?

## 최종 결과물

## 입력

## 출력

## 필요한 도구

## 폴더 구조

## 만들 기능

## 첫 번째 작업

## Codex에게 다음에 요청할 말

## 확인 방법

## 메모
```

## 작성 기준

- 입력과 출력이 눈에 보이게 씁니다.
- 첫 버전은 1시간 안에 확인 가능한 크기로 줄입니다.
- 외부 API, 로그인, 배포는 기본안으로 넣지 않습니다.
- 다음 날 다시 열어도 무엇을 해야 할지 알 수 있게 씁니다.
- 예시 파일이 필요하면 `input/`에 작고 안전한 샘플을 둡니다.
